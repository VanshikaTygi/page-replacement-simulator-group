# Page Replacement Simulator

This project is an interactive OS-based simulator to visualize different page replacement algorithms.

## 🚀 Features
- Visual simulation of page replacement algorithms
- Supports multiple algorithms:
  - FIFO (First In First Out)
  - LRU (Least Recently Used)
  - LFU (Least Frequently Used)
  - MFU (Most Frequently Used)
  - Optimal Algorithm
  - Clock Algorithm
- Clean UI with real-time updates

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📁 Project Structure
- index.html → Main UI
- style.css → Styling
- script.js → Core logic
- app.js → App control
- algorithms/ → Contains all algorithm implementations

## 🎯 Purpose
This project helps students understand how different page replacement algorithms work in Operating Systems.
