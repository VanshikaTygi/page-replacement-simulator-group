# 🚀 Page Replacement Simulator

An interactive Operating Systems project that visualizes how different page replacement algorithms work in memory management.

This simulator helps in understanding how pages are loaded, replaced, and managed in real-time, making complex OS concepts easier to grasp.

---

## 📌 Features

* 📊 Visual simulation of page replacement algorithms
* ⚡ Real-time updates of memory frames
* 🔁 Supports multiple algorithms:

  * FIFO (First In First Out)
  * LRU (Least Recently Used)
  * LFU (Least Frequently Used)
  * MFU (Most Frequently Used)
  * Optimal Algorithm
  * Clock Algorithm
* 🎯 Displays page faults and performance comparison

---

* HTML

## 📂 Project Structure

* `index.html` → Main user interface
* `style.css` → Styling
* `script.js` → Core logic
* `app.js` → Application control
* `algorithms/` → Contains implementations of all algorithms

---

## ⚙️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Enter page reference string and number of frames
4. Select algorithm and run simulation

---

## 📊 Output / Simulation

The simulator displays:

* Step-by-step page allocation
* Page faults
* Frame updates in real time
